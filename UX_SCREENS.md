# TuesdayOps UX Screens

## Design direction

The UI should feel:

- clean
- calm
- premium
- agency-friendly
- operational, not technical-heavy

Visual style:

- soft white/off-white background
- muted lavender primary accent
- dark slate typography
- restrained status colors
- rounded cards
- subtle borders/shadows
- generous whitespace

Avoid:

- cyberpunk dark UI for the main product
- too many colors
- overstuffed dashboards
- technical trace views in MVP
- heavy AI buzzwords

## Screen 1 — Landing Page

Goal: validate demand and collect calls.

Sections:

- Hero
- Product screenshot
- Problem
- How it works
- Sample report
- Pricing / validation call CTA

Primary CTA:

- Book a validation call

Secondary CTA:

- View sample report

## Screen 2 — Onboarding Wizard

Steps:

1. Create agency workspace
2. Add branding
3. Add first client
4. Add first workflow
5. Configure first check
6. Run first check
7. Generate sample report

Success state:

- “Your first workflow is now monitored.”
